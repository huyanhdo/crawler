# crawler
## cài đặt
```code
git clone https://github.com/huyanhdo/crawler.git
cd crawler
mkdir data
pip install virtualenv
python -m venv myenv
myenv/Scripts/activate
pip install -r requirements.txt
python crawler.py
```
